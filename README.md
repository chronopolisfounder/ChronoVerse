# ChronoVerse

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Run tests:

```bash
npm run test
```

## Features

- React 18 + Tailwind CSS frontend
- Context state management for Avatar, DreamFusion, Timeline
- Basic diagnostic loop integration
- React Router v6 setup
- Jest testing for core lib functions
